Juni 18, 2024 door Niels

Deze website is opgebouwd met HTML en voor de opmaak heb ik gebruik gemaakt van Bootstrap. Bootstrap is een populair CSS-framework dat het schrijven van CSS overbodig maakt door het gebruik van class tags. Het biedt een ruime keuze aan kant-en-klare stijlen en componenten, wat het eenvoudiger maakt om een professionele en responsive website te creëren. Ik heb gekozen voor een blogwebsite als uitgangspunt, geïnspireerd door de voorbeelden die Bootstrap biedt.
